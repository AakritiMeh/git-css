CSS TEXTS AND FONTS

Typography

Typography is the way that text is arranged and presented

Text Settings

text-transform ::: uppercase lowecase capitalize

text-decoration :::

underline(line below the text)
overline(line over the text)
line-through(cut thru the text)

text-align ::: left justify(same spacing on left and right) right

text-indent ::: 2ex(here we generally use em)-it will indent the first line of the paragraph

line-height ::: default=1.2 (difference btwn two lines)

letter-spacing ::: difference between two letter-spacing

word-spacing ::: difference between two words

font-weight ::: default=normal(300-400)

font-style ::: italic oblique normal

font-family ::: serif, sans-serif, monospace,cursive,fantasy....etc;
if we give multiple font fam then it will go in order,ie
if 1st is not available then 2nd if 2nd not available then 3rd..and so on

websafe fonts ::: that dont need to be installed externally and are mostly present on the system and web

for getting fonts from google,u can either import or paste link into html and css page
